## 📋 Описание
Веб-приложение для организации личного и командного времени. Позволяет создавать задачи, распределять их по статусам (Kanban-доска), назначать приоритеты и отслеживать выполнение.

### Основные возможности:
- 🔐 **Аутентификация**: Регистрация и авторизация пользователей.
- 📊 **Kanban-доска**: Интерактивная доска с Drag-and-Drop перемещением задач.
- 🎨 **Темы оформления**: Светлая и Темная (Cyberpunk) темы.
- ⚡ **Real-time**: Мгновенное обновление статусов.
